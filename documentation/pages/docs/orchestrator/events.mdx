# Orchestrator Messaging Events

## Text Formatting

Following event types support formatting:

+ text message
+ button template
+ generic template (card title and subtitle)

You can use this proprietary syntax to achieve different text formats:

- **Bold text**

    `**example**`

    > **example**

- **Italics text**

    `__example__`

    > *example*

- **Links**

    full links (with http/https schemes) are automatically converted to shortened form

    `https://wingbot.ai/sasalele?some=query#with=hash`

    > [wingbot.ai/sasalele](https://wingbot.ai/sasalele?some=query#with=hash)

- **Email links**

    `sasalele@wingbot.ai`

    > <sasalele@wingbot.ai>

- **Phone numbers**

    with country prefix

    `+420 602 123 456`

    > [+420 602 123 456](tel:+420602123456)

    with country prefix and some spacing characters

    `+420 (0609) 112-777`

    > [+420 (0609) 112-777](tel:+4200609112777)

    numbers without country code should be prefixed with `_tel:`

    `_tel:12345`

    > [12345](tel:12345)
