### buttonTemplate.urlButton(title, linkUrl, hasExtension, [webviewHeight], [onCloseAction], [onCloseData]) ⇒ <code>this</code>
Adds button. When `hasExtension` is set to `true`, url will contain hash like:
`#token=foo&senderId=23344`

**Kind**: instance method of [<code>ButtonTemplate</code>](#ButtonTemplate)  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| title | <code>string</code> |  | button text |
| linkUrl | <code>string</code> |  | button url |
| hasExtension | <code>boolean</code> | <code>false</code> | includes token in url |
| [webviewHeight] | <code>string</code> | <code>null</code> | compact|tall|full |
| [onCloseAction] | <code>string</code> | <code>null</code> | close action for webview |
| [onCloseData] | <code>object</code> |  | data |

<div id="ButtonTemplate_attachmentButton">&nbsp;</div>

### buttonTemplate.attachmentButton(title, payload) ⇒ <code>this</code>
Adds button, which opens a popup with content on click.

**Kind**: instance method of [<code>ButtonTemplate</code>](#ButtonTemplate)  

| Param | Type |
| --- | --- |
| title | <code>string</code> | 
| payload | [<code>Payload</code>](#Payload) | 

**Example**  
```js
res.button('text')
 .attachmentButton('button title', {
     content: '# Heading 1',
     contentType: 'text/markdown',
     className: 'my-class-for-markdown'
  })
 .send();
```
